#  Hero.app

##  Description
Hero.app is a modern web application designed to showcase and manage hero-related data. It features a clean and responsive UI, smooth navigation, and efficient performance for browsing heroes, their abilities, and profiles.

## Technologies Used
- **React.js** – Frontend framework for building interactive UIs  
- **Tailwind CSS** – Utility-first CSS framework for styling  
- **Node.js** – Backend runtime environment  
- **Express.js** – For handling APIs and server logic  
- **MongoDB** – NoSQL database for storing hero data  
- **Vite** – Fast development and build tool
